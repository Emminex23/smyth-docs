The `RAG Forget` component deletes data sources indexed by the `Data Source Indexer`.

> **Note:** Manually created data sources cannot be deleted by this component.

## Settings

### üìÇ Namespace

- **Namespace**: Select the namespace to identify and delete data sources.

### üîç Source Identifier

- **Source Identifier**: Specify the identifier of the data source to be deleted. Accepts input variables for dynamic identification.

---

## Inputs

The `Source Cleaner` component does not have default inputs.

### ‚ûï Add Input

- **Name (Required):** Unique, short, and self-explanatory.
- **Type:** Select data type (Any, String, Number, Integer, Boolean, Array, Object, Binary, Date).
- **Color (Optional):** Adds visual clarity.
- **Optional (Optional):** Mark as optional if not required.
- **Default Value (Optional):** Predefined value if none is provided.

---

## Outputs

### Success

- **Success**: 
  - Returns `true` if the data source is successfully deleted.
  - Returns an empty value if the deletion fails.
