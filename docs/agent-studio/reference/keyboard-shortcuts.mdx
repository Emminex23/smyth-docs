# Keyboard Shortcuts

Keyboard shortcuts help you move faster, think clearer, and stay in flow. Learning these keyboard shortcuts will help you spend less time clicking and more time building your agents.

<TipCallout title="Power user workflow">
Start your day with `⌘ + P` or `Ctrl + P` to jump into any agent, tweak components with `⌘ + Click`, copy with `⌘ + C`, and finish with a confident export.
</TipCallout>

<Divider />

## Canvas Navigation

Quickly move around the canvas and select components with precision.

| Action                     | Windows/Linux     | macOS             |
|----------------------------|-------------------|-------------------|
| Zoom in/out                | `Ctrl + Scroll`   | `⌘ + Scroll`      |
| Pan (drag view)            | `Scroll`          | `Scroll`          |
| Pan horizontally           | `Shift + Scroll`  | `Shift + Scroll`  |
| Select single component    | `Ctrl + Click`    | `⌘ + Click`       |
| Select multiple components | `Ctrl + Drag`     | `⌘ + Drag`        |

## Canvas Search

Quickly find and jump to components or agents using the in-canvas search.

| Action            | Windows/Linux | macOS       |
|-------------------|---------------|-------------|
| Open search menu  | `Ctrl + P`    | `⌘ + P`     |

<TipCallout title="Smart search">
This command lets you jump to any component on the canvas, including by name. It's the fastest way to find your place in large agents.
</TipCallout>

## Workflow Actions

Manage canvas components directly with these core editing commands.

| Action        | Windows/Linux | macOS           |
|---------------|----------------|------------------|
| Move item     | `Drag`         | `Drag`           |
| Copy          | `Ctrl + C`     | `⌘ + C`          |
| Paste         | `Ctrl + V`     | `⌘ + V`          |
| Delete        | `Del`          | `Delete` / `Fn + ⌫` |
| Delete connection      | `Click on wire`   | `Click on wire`   |

## Power User Tips

- Use `⌘ + P` / `Ctrl + P` early and often. It’s the fastest way to stay oriented.
- Build small sections, copy/paste when patterns emerge.
- Get used to `Undo`/`Redo` because no fear of mistakes makes you bolder.
- Keyboard shortcuts stack well with zoom and drag for fast editing.
