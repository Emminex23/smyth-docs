# Logs

The **Logs** feature in SmythOS provides comprehensive visibility into your agents' activities, helping you monitor performance, debug issues, and track usage.

<InfoCallout>
Use logs to understand how your agents are performing, identify bottlenecks, and troubleshoot any issues that arise.
</InfoCallout>

<Spacer size="md" />

## Viewing Logs

Access your agent's logs through:

1. The **Agent Dashboard** → Select an agent → Click "Logs"
2. The **Builder Canvas** → Click "Logs" in the top toolbar
3. The **Agent Settings** → Navigate to the "Logs" tab

<Spacer size="md" />

## Log Types

SmythOS captures several types of logs:

| Log Type | Description | Use Case |
|----------|-------------|-----------|
| **Execution Logs** | Detailed step-by-step records of agent runs | Debug workflow issues |
| **Performance Logs** | Timing and resource usage metrics | Optimize agent speed |
| **Error Logs** | Detailed error messages and stack traces | Troubleshoot failures |
| **API Logs** | Records of external API calls | Monitor integrations |
| **Security Logs** | Authentication and access attempts | Audit security |

<Spacer size="md" />

## Log Retention

- **Free Plan**: 7 days of logs
- **Pro Plan**: 30 days of logs
- **Enterprise Plan**: Custom retention period
