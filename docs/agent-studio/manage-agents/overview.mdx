## Configuration

### Identity
- • **Name** - Unique identifier for your agent
    - Choose descriptive names for easy recognition
    - Used in UI and API interactions
- • **Avatar** - Visual representation
    - Auto-generate based on name
    - Upload custom image (recommended: 256x256px)
- • **Description** - Brief summary of agent's purpose

### Behavior Settings
- • **Default LLM** - Primary language model
    - Used for chat and chatbot interactions
    - Affects response quality and capabilities
- • **Intro Message** - Initial chatbot greeting
    - Customizes user's first interaction
    - Sets expectations for agent capabilities
- • **Behavior Description** - Detailed functionality guide
    - Acts as system prompt for the LLM
    - Defines agent's personality and responses

## Global Variables

Variables accessible throughout all your agent's components. Changes to global scope reflect immediately in workflow wherever they are used.

### Common Uses
- • API endpoints
- • Configuration values
- • Shared constants

> **Security Note**: Store sensitive information like API keys in the [Vault](/docs/agent-collaboration/working-with-agents/vault) instead of global variables.
