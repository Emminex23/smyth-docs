# JSON Filter

The `JSON Filter` component streamlines extensive API responses by filtering out irrelevant information, preventing agent failures due to context window limitations.

---

## Settings

- **Filter Parameters:** List parameters to retain, including all parent objects for nested parameters.
  > **Note:** Always include parent parameter names for accurate filtering.

---

## Inputs

- **Input:** Field for the JSON object to be filtered.

---

## Outputs

- **Output:** Field for the filtered JSON object.
