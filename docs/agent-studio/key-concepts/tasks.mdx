# Tasks

The **Tasks** feature in SmythOS allows you to schedule and automate your agents' workflows, ensuring they run at the right time and frequency.

<InfoCallout>
Use tasks to automate repetitive workflows, schedule regular maintenance, and ensure your agents run when needed.
</InfoCallout>

<Spacer size="md" />

## Creating Tasks

1. Go to **Agent Settings** â†’ **Tasks**
2. Click **Add Task**
3. Configure:
   - Schedule (cron expression)
   - Input parameters
   - Error handling
   - Notifications

<Spacer size="md" />

## Task Types

| Task Type | Description | Use Case |
|-----------|-------------|-----------|
| **Scheduled** | Run at specific times | Daily reports, maintenance |
| **Interval** | Run every X minutes/hours | Regular data sync |
| **Event-based** | Trigger on specific events | Webhook responses |
| **Manual** | Run on demand | Testing, one-off tasks |

<Spacer size="md" />

## Task Management

- **View History**: See past runs and their status
- **Monitor Performance**: Track execution time and resource usage
- **Handle Errors**: Configure retry logic and notifications
- **Pause/Resume**: Temporarily stop tasks without deleting them

<Spacer size="md" />

## Best Practices

1. **Start Simple**: Begin with basic schedules
2. **Test Thoroughly**: Verify task behavior before production
3. **Monitor Closely**: Watch for errors and performance issues
4. **Use Notifications**: Get alerts for important events
5. **Document Tasks**: Keep track of what each task does
