# ForEach Component

The `ForEach` component in SmythOS processes array elements sequentially, enabling efficient data handling and transformation.

---

## Settings

### Output Format
1. **Full:** Detailed results for comprehensive analysis.
2. **Minimal:** Streamlined results for quick overviews.
3. **Array of Results:** Structuring results for data manipulation.

---

## Inputs

- **Array Format:** Input must be an array.
- **Data Type:** Any data type.
- **External Sources:** Can be sourced from user interactions, other systems, or integrations.

---

## Outputs

### Loop
- Processes each array element sequentially.

### Result
- Aggregates outcomes of all iterations.

> **Note:** Bind a component to this output to use aggregated results.
