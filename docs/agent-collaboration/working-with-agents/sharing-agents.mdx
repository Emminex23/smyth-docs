# How To Share an Agent

Easily share and provide access to your agent using the Share Agent feature.

### Bulk Sharing

- **➕ Add Multiple Emails**: Share the agent with multiple users by adding multiple email addresses at once.
- **🔄 Compatibility**: Share with users already part of your team or new users. Users can have a SmythOS account or not.

### Sharing Across Spaces

- **🌐 Different Spaces**: If the agent is in a different space, the invited user will also be added to that space.

### Notifications

- **📧 Email Notification**: All invited email addresses will receive an email with instructions on accessing the agent.

By using the Share Agent feature, you can efficiently grant access to your agents to a wide range of users and ensure they receive the necessary notifications and instructions.
