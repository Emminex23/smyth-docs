# DataForSEO Integration

The `DataForSEO` API offers a robust foundation for creating sophisticated SEO software. It provides comprehensive data that helps in constructing efficient applications, optimizing your time and budget management significantly.

The integration with `SmythOS` was implemented to enhance its SEO capabilities and tools, allowing users to access precise and up-to-date SEO data quickly. This integration aids in automating and refining SEO strategies within `SmythOS`, making it a more powerful tool for digital marketing professionals.

## Live Google SERP Regular

This component allows for live querying of Google's Search Engine Results Pages (SERP), using the `DataForSEO` API to provide real-time search result data. It is perfect for applications requiring up-to-the-minute data for SEO analysis and competitive insight.

### Settings

#### Base64 Encoded Login and Password

For activation, encode your login and password in Base64 in the format `login:password`. Use the encoded API credentials instead of plain text.

For example, a Base64-encoded `login:password` appears as: `bG9naW46cGFzc3dvcmQ=`.

To encode your credentials:

1. Log into your `DataForSEO` account and access your [API Dashboard](https://app.dataforseo.com/api-dashboard).
2. Encode your login and password in Base64.
3. Input the encoded values into the designated settings section of the component.

> **Note:** Treat the `login` and `password` as sensitive data. In `SmythOS`, you can securely manage it using the `Vault` feature. For more details, please refer to [this](https://app.smythos.com/doc/#/vault/keysManagement) section in the documentation.

<Image src="/img/dataforseo/screenshot1.png" alt="alt text" /> 

### Inputs

Customize your search with the following configurable inputs:

1. **language_name `optional<string>`** (default: `english`): Specifies the language for the search.
2. **location_name `optional<string>`** (default: `united states`): Sets the geographic location for the search.
3. **keyword `required<string>`**: Indicates the search keyword or phrase.

<Image src="/img/dataforseo/screenshot2.png" alt="alt text" />

> **Note:** Like many components of `SmythOS`, you can extend this list by adding new inputs with the `+` button in the `Inputs` section.

### Outputs

The component produces the following outputs for detailed data analysis:

1. **Response**: The primary JSON response containing all search data.
2. **Headers**: Captures HTTP headers from the response, useful for debugging and compliance checks.
3. **id**: A unique identifier for the search.
4. **result**: The search results derived from the query.
5. **items_count**: The number of items returned in the search results.

<Image src="/img/dataforseo/screenshot3.png" alt="alt text" />

> **Note:** Similar to inputs, you can add new outputs by clicking on the `+` button in the `Outputs` section. This feature can come in handy when you aim to extract specific values from the `Response`. Please refer [here](https://app.smythos.com/doc/#/components/APICall?id=example-without-authentication) to see a practical example.
