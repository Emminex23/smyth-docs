# Keyboard Shortcuts

Master the SmythOS Studio canvas with keyboard shortcuts. This guide covers all the essential hotkeys to help you build, navigate, and manage your agent workflows faster.

<InfoCallout>
**Pro Tip:** Use these shortcuts to build and debug agents more efficiently, reducing clicks and keeping you in the creative flow.
</InfoCallout>

<Divider />

## Navigation

Use these shortcuts to move around the canvas, zoom, and select components with ease.

| Action             | Windows/Linux   | macOS            |
| ------------------ | --------------- | ---------------- |
| Zoom               | `Ctrl + Scroll` | `⌘ + Scroll`     |
| Pan                | `Scroll`        | `Scroll`         |
| Pan Horizontal     | `Shift + Scroll`| `Shift + Scroll` |
| Select (Single)    | `Ctrl + Click`  | `⌘ + Click`      |
| Select (Multiple)  | `Ctrl + Drag`   | `⌘ + Drag`       |

## Workflow Actions

Perform common actions on components like moving, copying, and deleting.

| Action | Windows/Linux | macOS               |
| ------ | ------------- | ------------------- |
| Move   | Drag          | Drag                |
| Copy   | `Ctrl + C`    | `⌘ + C`             |
| Paste  | `Ctrl + V`    | `⌘ + V`             |
| Delete | `Del`         | `Delete` / `Fn + ⌫` |

## Import and Export

Quickly save or load entire agent workflows using these shortcuts.

| Action | Windows/Linux      | macOS              |
| ------ | ------------------ | ------------------ |
| Export | `Ctrl + Shift + E` | `⌘ + Shift + E`    |
| Import | Drag `.smyth` file | Drag `.smyth` file |

## Version Control

Instantly undo or redo changes to your workflow. SmythOS tracks your changes so you can experiment safely.

| Action | Windows/Linux      | macOS           |
| ------ | ------------------ | --------------- |
| Undo   | `Ctrl + Z`         | `⌘ + Z`         |
| Redo   | `Ctrl + Y`         | `⌘ + Shift + Z` |
