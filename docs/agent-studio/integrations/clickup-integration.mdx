# ClickUp Integration

## Introduction

The ClickUp Integration for SmythOS brings the power and flexibility of ClickUp's task management and productivity tools directly into the SmythOS environment. By integrating ClickUp into SmythOS, users can streamline their workflows, manage tasks effectively, and enhance productivity without switching between platforms.

ClickUp is a versatile project management and productivity platform widely used for its robust features, including task management, goal tracking, time management, and collaboration tools. The ClickUp API enables developers to build custom integrations and applications that interact with the ClickUp platform, providing extensive functionalities that can be tailored to specific needs.

The ClickUp Integration is a valuable addition to SmythOS, offering powerful task management and productivity features from ClickUp directly within the platform. By leveraging the capabilities of the ClickUp API, users can enhance their workflow, stay organized, and boost productivity seamlessly.

For more detailed information, please visit the [ClickUp API documentation page](https://clickup.com/api).

## Add Task

### ⚙️ Settings

#### 🔑 ClickUp API Token

To activate the Add Task component, you need to obtain your `ClickUp API Token` by following these steps:

1. **🔐 Log In to Your ClickUp Account**
   - Navigate to the [ClickUp website](https://clickup.com) and log in with your credentials.

2. **👤 Access Your Profile**
   - Click on your avatar or profile picture in the lower-left corner to open the User Menu.

3. **🖥️ Navigate to Apps**
   - From the User Menu, select **Apps**. This will take you to the API Token management page.

4. **🆕 Generate API Token**
   - If you haven't already generated a token, click the **Generate** button. If a token already exists, you can copy it from this page.

5. **📋 Copy the API Token**
   - Copy the `API Token` that you generated. This token uniquely identifies your account and grants access to the ClickUp API.

6. **✍️ Enter the API Token in SmythOS**
   - Paste this `ClickUp API Token` into the appropriate field in the component's settings within SmythOS.

> **Note:** Treat the `ClickUp API Token` as sensitive data. In `SmythOS`, you can securely manage it using the `Vault` feature. For more details, please refer to [this](https://app.smythos.com/doc/#/vault/keysManagement) section in the documentation.

<Image src="/img/clickup/screenshot1.png" alt="alt text" /> 

### 📥 Inputs

One default input is necessary to execute this component:

1. **🔗 `list_id` ``` _required<string>_: ```** The unique identifier of the list where the task will be added.
2. **📝 `name` ```_required<string>_:```** The name of the task.
3. **📝 `description` ```_optional<string>_:```** A detailed description of the task.
4. **📊 `priority` ```_optional<integer>_:```** The priority level of the task, with a default value of `3`.

<Image src="/img/clickup/screenshot2.png" alt="alt text" />

> **Note:** With the flexibility and customization in mind, `SmythOS` allows you to expand the default input list by adding new inputs using the `+` button situated in the `Inputs` section.

### 📤 Outputs

After executing this component, you will receive the following outputs:

1. **📜 `Response`:** A detailed description of the response received from the platform.
2. **📄 `Headers`:** Specific headers returned during the request.
3. **🆔 `id`:** The ID of the created task.
4. **🌐 `url`:** The URL of the created task.
5. **🔗 `list_id`:** The list ID associated with the created task.
6. **📁 `folder_id`:** The folder ID associated with the created task.
7. **🏠 `space_id`:** The space ID associated with the created task.

<Image src="/img/clickup/screenshot3.png" alt="alt text" />

> **Note:** Similar to inputs, you can add new outputs by clicking on the `+` button in the `Outputs` section. This feature can come in handy when you aim to extract specific values from the `Response`. Please refer [here](https://app.smythos.com/doc/#/components/APICall?id=example-without-authentication) to see a practical example.
