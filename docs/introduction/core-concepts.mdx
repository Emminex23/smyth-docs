| Term | Meaning in SmythOS |
|------|-------------------|
| **Agent** | A workflow that takes input, calls tools, returns output |
| **Component** | A single step (LLM, API call, Python snippet) |
| **Vault** | Encrypted store for API keys and secrets |
| **Skill** | Reusable group of components, importable into any agent |
| **Runtime** | Engine that executes agents in the cloud or locally |
