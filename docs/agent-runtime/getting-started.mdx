# Getting Started with SRE

<InfoCallout title="What you'll learn">
Install the Smyth Runtime, configure a local dev setup, and run your first test agent using the SDK.
</InfoCallout>

<Spacer size="md" />

## 1. Install the runtime

Install the package via npm or clone the [GitHub repo](https://github.com/SmythOS/smyth-runtime/tree/os-code-restructure) ):

```bash
npm install smyth-runtime 
```

## 2. Initialize the runtime

Import and initialize SRE in your project. Use a simple configuration for local development:

```typescript
import { SRE, SDK } from 'smyth-runtime';

const sre = SRE.init({
  Cache: { Connector: 'RAM' },
  Storage: { Connector: 'Local' },
  Log: { Connector: 'ConsoleLog' },
});

await sre.ready();
```
This sets up in-memory caching, local file storage, and console logging for you, which is a good starting point.

## Create an LLM instance

Use the built-in SDK to test if the LLM is working:

```typescript
import { SDK } from 'smyth-runtime';

const llm = new SDK.LLM('OpenAI', {
    model: 'gpt-4',
    temperature: 0.7,
});
const res = await llm.prompt('Explain quantum computing to a medical student');
console.log(res);
```

## Stream results (optional)

Useful for chat or interactive apps:

```typescript
const stream = await llm.prompt('Write a short story').stream();
stream.on('content', (chunk) => process.stdout.write(chunk));
```
