## Prerequisites

* A SmythOS account (sign up free)
* Two minutes of curiosity

## Step 1. Create a workspace

1. Log in  
2. Click **New Workspace**  
3. Name it **demo-bots**

## Step 2. Launch Agent Studio

Click **Create Agent**.  
You will see an empty canvas.

## Step 3. Add an LLM component

1. Drag **OpenAI** from the sidebar to the canvas  
2. Paste your API key into the **Vault** prompt

## Step 4. Connect input and output

Draw a line from **Start** to **OpenAI** then to **End**.

Hit **Run**. The agent echoes your prompt.
