## Settings

### Output Format

Customize data presentation:

1. **Full (Default):** Comprehensive view with component IDs, names, and descriptions. 
   - **Ideal for:** In-depth analysis.
  
2. **Minimal:** Displays only essential information.
   - **Ideal for:** Simplicity and speed.
  
3. **Raw:** Outputs unprocessed data in a JSON object.
   - **Ideal for:** Custom handling or processing.

---

## Inputs

Inputs guide data into your workflow, forming the basis of the output.

- **Name (Required):** Unique and illustrative.
- **Type:** Choose from `Any`, `String`, `Number`, `Integer`, `Boolean`, `Array`, `Object`, `Binary`, `Date`.
- **Color (Optional):** Differentiates inputs visually.
- **Optional (Optional):** Mark non-mandatory inputs as optional.
- **Default Value (Optional):** Ensure smooth operation with default values.

---

## Outputs

Provides a single, consolidated output named `Output`, organizing all inputs into a structured JSON format.
