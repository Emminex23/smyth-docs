# Quickstart Guide

Use this quickstart to get a working agent up and running in SmythOS in just a few steps. You'll use the [Lead Enrichment Agent](/docs/agent-templates/sales/lead-enrichment-agent) as your starting point. It collects public data to enrich lead profiles with actionable insights.

<Youtube videoId="V0ft3GrLtjY" title="SmythOS - Lead Enrichment Agent" />

<Spacer size="md" />

## Step 1: Remix the Lead Enrichment Agent

To start, you'll remix a prebuilt agent from the Templates section:

1. Go to **[Templates](https://app.smythos.com/templates)** from the top navigation bar.
2. Open the **Sales** tab.
3. Find the **Lead Enrichment Agent** and click **Remix**.

<Image src="/docs/img/lead_enrichment_agent/img_1.png" alt="Navigate to Templates" />
<Image src="/docs/img/lead_enrichment_agent/img_2.png" alt="Remix Agent" />

<Spacer size="md" />
## Step 2: Run the Agent with Form Preview

You can test your agent using either of these methods:

**Option 1: From the top toolbar**

1. Click **Test** in the top-right toolbar.
2. Switch to the **Form Preview** tab.
3. Fill the form details.
4. Click **Run** to execute the workflow.

<Spacer size="md" />

**Option 2: From the canvas**

1. Click the **Form Preview** button on the relevant block.
2. Enter the same input fields and fill the form.
3. Click **Run** to test and view results.

## Step 3: Test APIs Using Mock Data (Optional)

No API keys? Use mock data to test workflows with API Call components [API Call components](/docs/agent-studio/components/advanced/api-call).

For more details, visit [How to set up mock data](/docs/agent-studio/components/advanced/api-call#step-5-enable-mock-mode)

<Spacer size="md" />

## Step 4: Deploy the Agent

Ready to go live? You can [deploy](/docs/agent-deployment/overview) directly from the Studio.

1. Click **Deploy** (top-right).
2. Choose your environment: Cloud, Dev, or Custom.
3. Monitor using Agent [**Logs**](/docs/agent-studio/manage-agents/logs).

Want to configure custom domains? See [Subdomains Guide](/docs/agent-deployments/subdomains).
