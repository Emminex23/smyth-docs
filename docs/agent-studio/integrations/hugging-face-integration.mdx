# Hugging Face Integration

<Arcade  src="https://demo.arcade.software/ASBAREF6ADkV0dNT6aFR?embed&embed_mobile=tab&embed_desktop=inline&show_copy_link=true" title="Hugging Face | SmythOS" />

---

Harness the power of `Hugging Face` within `SmythOS`, bringing a variety of AI-trained models directly to the platform. These models excel in areas such as advanced language understanding, image recognition, audio processing, and sentiment analysis.

---

## Settings

To use Hugging Face models, an **Access Token** is required. Obtain a free Access Token by creating a Hugging Face account.

1. **🛡️ Get Access Token**: Visit the [Hugging Face Tokens page](https://huggingface.co/settings/tokens).
2. **✅ Validate in SmythOS**: Copy and paste the Access Token into the `Access Token` field in the Hugging Face component.

---

## Inputs & Outputs

- **🔡 Inputs**: Vary based on the chosen model. Refer to the model's documentation for details.
- **🔍 Outputs**: Displayed in the `Output` field, showing results processed by the plugin.
