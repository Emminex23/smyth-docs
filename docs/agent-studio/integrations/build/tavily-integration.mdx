# Tavily Integration

Tavily is an AI-powered platform designed to streamline data collection and utilization. It specializes in scraping, filtering, and aggregating information from over 20 web sources, making it indispensable for applications requiring comprehensive data retrieval and insightful analytics.

Integrating Tavily into SmythOS provides numerous advantages that enhance the overall functionality of the system:

1. **Enhanced Data Access**: Tavily’s capabilities allow SmythOS to access comprehensive and relevant data from a multitude of sources, ensuring users have the most accurate and up-to-date information.
2. **AI-Powered Efficiency**: The platform leverages artificial intelligence to optimize data retrieval and utilization, improving the system's overall efficiency and effectiveness.
3. **Advanced Search Functionalities**: Tavily’s sophisticated search tools empower SmythOS to offer more detailed and relevant search results, enhancing user experience.
4. **Seamless Integration**: Tavily is designed for easy integration, allowing SmythOS to quickly implement new features and improve existing ones without significant overhead.

By leveraging Tavily, SmythOS can deliver a more robust, efficient, and user-friendly experience, maintaining its competitive edge in the market.

## Integratin Settings
To use Tavily integration in `SmythOs`, you must authenticate each integration component with a Tavily API key. Follow these steps to prepare your credentials:

#### Tavily API Key
To initialize and configure the `Tavily` integration components, follow these simple guidelines to obtain and input your Tavily API key:
1. **Get Your Tavily API Key:**
   - **Sign In:**
     - Go to the Tavily sign-in page: [Tavily Sign-In](https://app.tavily.com/sign-in) and click on "Sign In".
     - Choose a sign-in method, such as signing in with Google.
     - Select one of your Google accounts to proceed.
   - **API Key Access:**
     - After signing in, you will be redirected to the Tavily homepage.
     - Navigate to the "API Key" section.
     - Copy your API key from there. Keep it confidential and do not share it publicly.

> **Note:** Treat the `Tavily API Key` as sensitive data. In `SmythOS`, you can securely manage it using the `Vault` feature. For more details, please refer to [this](https://app.smythos.com/doc/#/vault/keysManagement) section in the documentation.

## Basic General Search
The Tavily Basic General Search engine is optimized for Large Language Models (LLMs) and Retrieval-Augmented Generation (RAG). It provides efficient, quick, and persistent general topic search results, making it an invaluable tool for data-driven applications.

### Component Settings
To use this component:
- Click on the component settings.
- Add your `Tavily API Key` from the `vault`. Please refer to the [Integration Settings](#integration-settings) to learn how to create it.
- Confirm and close the settings.

### Inputs
To effectively utilize the capabilities of the `Basic General Search` component, ensure the provision of the following input parameters:
1. **query `required<string>`:** The search query string required to retrieve relevant results.
2. **max_results `optional<string>`:** Maximum number of results to return. Defaults to a predefined number if not specified.
3. **include_domains `optional<array>`:** List of domains to include in the search results.
4. **exclude_domains `optional<array>`:** List of domains to exclude from the search results.

> **Note:** With the flexibility and customization in mind, `SmythOS` allows you to expand the default input list by adding new inputs using the `+` button situated in the `Inputs` section.

### Outputs
Upon successful execution of the `Basic General Search` component, expect to receive the following outputs:
1. **Response:** An object containing comprehensive details regarding the operation. This output includes information about the search results, such as their source, relevance, and any other metadata.
2. **Headers:** HTTP headers extracted from the server response provided by Tavily.
3. **Results:** Detailed search results returned by the Tavily API.

> **Note:** Similar to inputs, you can add new outputs by clicking on the `+` button in the `Outputs` section. This feature can come in handy when you aim to extract specific values from the `Response`. Please refer [here](https://app.smythos.com/doc/#/components/APICall?id=example-without-authentication) to see a practical example.
