## Creating a Data Space

1. **Click `Add Data Space`** to open the create popup.
2. **Provide a unique name** for your Data Space.
3. **Toggle `Use Custom Storage`** (if set up):
    - **Disabled**: Index in SmythOS's default storage.
    - **Enabled**: Index in Custom Storage.
4. **Click `Add Data Space`** to finalize.

## Adding Data Sources

1. **Open the Data Space** and click `Add Data Source`.
2. **Provide Details**:
    - Label for the data source.
    - Select **Data Source** type:
        - **File**: Upload (.xml, .docx, .pdf).
        - **URL**: Use website URL or sitemap.

> **Note:** You can add multiple data sources as needed.

## Custom Storage

**Custom Storage** in SmythOS allows indexing in SmythOS's storage or external service [Pinecone](https://www.pinecone.io/).

### Setting Up Custom Storage

#### 1. Create an Index in Pinecone
- **Create Account**: Sign up on Pinecone.
- **Create Index**: Go to `Indexes` and click `Create Index`.
- **Assign Name & Dimensions**: Name the index and set dimensions to `1536`.
- **Select Capacity Mode**: Choose `PODS` or `SERVERLESS`.
- **Apply Changes**: Save.

#### 2. Integrate Pinecone with SmythOS
- **Configure Storage**: Go to `Data Pools` and click `Configure Storage`.
- **Retrieve API Key**: Get it from Pinecone's `API keys` page and paste it into SmythOS.
- **Get Index Details**: Obtain `Index Name`, `Region/Environment`, and `Index Host` from Pinecone.
- **Complete Setup**: Enter the required information.

#### 3. Use Custom Storage Option
- **Enable `Use Custom Storage`** option when creating Data Spaces.

Custom Storage with Pinecone provides better control and efficiency in data management for your agents.
