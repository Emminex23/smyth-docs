The `Sleep` component introduces a pause in a workflow, delaying operations for a specified duration, useful for process synchronization and timing.

---

## Settings

- **Delay:** Set the pause length in seconds (default: 1 second, max: 3600 seconds or 1 hour).

---

## Inputs

- **Input:** Activates the sleep function. The content of the input is not processed, but a value is required to trigger the delay.

---

## Outputs

- **Output:** An empty string indicating the end of the delay, signaling the workflow to proceed.
