# Fal.ai Integration

`Fal.ai` is a frontrunner in the domain of Artificial Intelligence (AI), offering robust solutions related to AI model deployment and execution. Leveraging Serverless GPU Cloud, Real-time AI applications, and Pre-trained AI Models, they strive to deliver high-performance AI inference.

Recognizing the immense potential of `Fal.ai`, we at `SmythOS` have designed custom components to seamlessly integrate `Fal.ai` services. This guide provides a detailed walkthrough on how to effectively utilize these components.

## Generate Animation

A standout feature of our `Fal.ai` integration is the **Generate Animation** component. This powerful tool enables users to create animations effortlessly and efficiently.

### ‚öôÔ∏è Settings

#### üîë Fal.ai API Key

To initialize this feature, you must first formulate and secure your `API Key` through the following steps:

1. Create a `Fal.ai` account if you don't already have one.
2. Navigate to your account control panel and generate your `API Key`. You can do this by clicking [here](https://fal.ai/dashboard/keys).
3. Input this `API Key` in the specified section in the component's settings.

> **Note:** Treat the `API Key` as sensitive data. In `SmythOS`, you can securely manage it using the `Vault` feature. For more details, please refer to [this](https://app.smythos.com/doc/#/vault/keysManagement) section in the documentation.

#### üìè Video Size

You can choose from the following video size options:

- **square_hd**
- **square**
- **portrait_4_3**
- **portrait_16_9**
- **landscape_4_3**
- **landscape_16_9**

#### ‚è≥ Steps

This parameter defines the number of steps for the model to run. A higher number results in better quality but may take longer to generate. The default value is set to **4**.

<Image src="/img/falai/screenshot1.png" alt="alt text" /> 

### üìù Inputs

To execute this feature, the following inputs are necessary:

1. **prompt (`required<string>`)**: The primary instruction or description for the animation.
2. **negative_prompt (`optional<string>`)**: Specifies undesired outcomes, with a default value of **(bad quality, worst quality:1.2), ugly faces, bad anime**.
3. **num_frames (`optional<integer>`)**: The number of frames to generate, default is **16**.
4. **fps (`optional<integer>`)**: Frames per second for the animation, default is **8**.

<Image src="/img/falai/screenshot2.png" alt="alt text" />

> **Note:** With the flexibility and customization in mind, `SmythOS` allows you to expand the default input list by adding new inputs using the `+` button situated in the `Inputs` section.

### üì§ Outputs

Upon successfully generating an animation, the following outputs will be available:

1. **Response**: Contains essential information about the created animation, including the video URL or file name.
2. **Headers**: Displays the HTTP headers from the response received from the **Fal.ai** server, providing insights into the characteristics of the transmitted data.
3. **video_url**: Direct URL link to the generated video, allowing immediate access and sharing options.

<Image src="/img/falai/screenshot3.png" alt="alt text" />

> **Note:** Similar to inputs, you can add new outputs by clicking on the `+` button in the `Outputs` section. This feature can come in handy when you aim to extract specific values from the `Response`. Please refer [here](https://app.smythos.com/doc/#/components/APICall?id=example-without-authentication) to see a practical example.
