The `Logic At Least` component sets the `Verified` output to `True` if at least the minimum number of inputs are set; otherwise, it is `No Value`.



---

## Settings

- **Minimum Inputs:** Define the minimum number of inputs (0 to 9) to trigger the `Verified` output.

---

## Inputs

### Boolean Values

- **True**: Any non-empty value.
- **False**: Specifically 'false' or unassigned (must be explicitly set as Boolean).

### âž• Add Input

- **Name (Required):** Unique, concise name.
- **Type:** Choose from Any, String, Number, Integer, Boolean, Array, Object, Binary.
- **Color (Optional):** For visual clarity and priority.
- **Optional (Optional):** Mark as optional if not required.
- **Default Value (Optional):** Predefined value if none is provided.

---

## Outputs

1. **Verified:** `True` if at least the minimum number of inputs are set; otherwise `No Value`.
2. **Unverified:** Inverse of `Verified`.
