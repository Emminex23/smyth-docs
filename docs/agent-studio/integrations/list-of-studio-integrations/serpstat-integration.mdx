# Serpstat

Serpstat is a comprehensive SEO and digital marketing platform that provides keyword research, competitor analysis, and site audit capabilities. It helps marketers and SEO professionals analyze search data, track competitors, and optimize their online presence. The platform offers extensive data analytics and insights for improving search engine rankings and digital marketing strategies.

At `SmythOS`, we have fully integrated `Serpstat` capabilities within our platform, providing custom components that facilitate advanced SEO research and competitive analysis. This integration allows users to access powerful keyword research tools, competitor analysis features, and detailed search analytics directly within their workflows.

## Search Suggestions

Provides search suggestions for the keyword you requested through full-text search functionality.

### Inputs
- **id (`required<string>`)**: Unique identifier for the request
- **keyword (`required<string>`)**: The search term to analyze
- **filters (`optional<object>`)**: Additional filtering parameters
- **se (`optional<string>`)**: Search engine and region (default: g_us)
- **page (`optional<integer>`)**: Page number for pagination (default: 1)
- **size (`optional<integer>`)**: Number of results per page (default: 100)

### Outputs
- **Response**: Complete API response
- **Headers**: Response headers
- **id**: Request identifier
- **result_data**: Search suggestion results
- **page**: Current page number
- **total**: Total number of results
- **left_lines**: Remaining available requests
