# Studio Interface

### Navigation

| Action             | Windows         | Mac              |
|--------------------|-----------------|------------------|
| Zoom               | `Ctrl + Scroll` | `⌘ + Scroll`     |
| Pan                | `Scroll`        | `Scroll`         |
| Pan Horizontal     | `Shift + Scroll`| `Shift + Scroll` |
| Select (Single)    | `Ctrl + Click`  | `⌘ + Click`      |
| Select (Multiple)  | `Ctrl + Drag`   | `⌘ + Drag`       |

### Workflow Actions

| Action | Windows     | Mac                 |
|--------|-------------|---------------------|
| Move   | Drag        | Drag                |
| Copy   | `Ctrl + C`  | `⌘ + C`             |
| Paste  | `Ctrl + V`  | `⌘ + V`             |
| Delete | `Del`       | `Delete` / `Fn + ⌫` |

### Import / Export

| Action | Windows             | Mac                 |
|--------|---------------------|---------------------|
| Export | `Ctrl + Shift + E`  | `⌘ + Shift + E`     |
| Import | Drag `.smyth` file  | Drag `.smyth` file  |

### Version Control

| Action | Windows     | Mac             |
|--------|-------------|-----------------|
| Undo   | `Ctrl + Z`  | `⌘ + Z`         |
| Redo   | `Ctrl + Y`  | `⌘ + Shift + Z` |

> Tracks changes so you can undo, redo, and experiment safely.
