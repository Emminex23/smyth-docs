# SmythOS Agents

The `SmythOS Agent` component allows you to reuse existing agents within other agents, creating more complex functionalities efficiently.

### 🌟 Key Benefits

- **♻️ Reusability**: Build advanced functionalities using existing agents.
- **🧩 Modularity**: Break down complex tasks into manageable components.
- **📈 Scalability**: Extend agent functionalities without duplicating efforts.

## Settings

### 🧬 Model

Choose an AI model to use:
- **OpenAI Models**: Default options available.
- **Other Models**: Add your keys in the [Vault](/docs/agent-collaboration/working-with-agents/vault-sharing)  to use other models.

### 🛠️ Version

Options for setting agent versions:
- **Same As Parent**: Uses the version (Dev/Prod) of the parent agent.
- **Dev**: Always uses the Dev version.
- **Prod Latest**: Always uses the latest Prod version.

### 📝 Description for Model

Default content from the subagent's `Behavior` field, which can be modified.

## Inputs

- **🔎 Prompt**: The default input required for the `SmythOS Agent` to function.

## Outputs

- **📊 Response**: Holds the agent's response, varying based on the agent's nature and specific operations.
